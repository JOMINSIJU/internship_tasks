### TASK 1
# Text Summarization Tool using NLP

## Overview
This project implements an extractive text summarization tool using Natural Language Processing techniques in Python.

## Features
- Sentence tokenization
- Stopword removal
- Word frequency analysis
- Automatic summary generation

## Technologies Used
- Python
- NLTK
- Jupyter Notebook

## Limitation
The model relies on word frequency and does not understand deep context or semantics.



### TASK 2
## Speech Recognition System

This project converts spoken audio into text using Python.

## Technologies Used
- Python
- SpeechRecognition Library
- PyAudio
- Google Speech API

## How it Works
1. The system captures audio through a microphone.
2. The SpeechRecognition library processes the audio.
3. A pre-trained speech model converts speech into text.
4. The recognized text is printed as output.
